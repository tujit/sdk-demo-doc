# JSON Web Token

The Authkit.io - JWT Decoder allows to decode JWT tokens. JSON Web Token (JWT) is a compact, URL-safe means of representing claims to be transferred between two parties.
It is often used for Authorization and Information Exchange in Identity & Access Management

[Learn more](https://docs.cidaas.com/docs/cidaas-iam/dxvjtbdxbwt08-validate-token) about JWT, JWS, and JWE.
