# Authorization Code Flow

The Authorization Code flow is the most secure way to implement OAuth 2.0 for server-side applications.

## How it Works

1. Your application redirects to Cidaas
2. User authenticates and authorizes your app
3. Cidaas redirects back with an authorization code
4. Your server exchanges the code for tokens
